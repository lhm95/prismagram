# prismagram

Instagram clone with Express + Prisma + React and React Native

npm install --save-dev @babel/cli @babel/core @babel/preset-env @babel/preset-stage-2

- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [x] Create account
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
